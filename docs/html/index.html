SQLite とは何か？

SQLite は、 小さくて、 高速で、 自己完結型で、 高信頼性で、 フル機能の SQL データベース・エンジンを実装する C 言語ライブラリです。 SQLite は、 世界で最も使用されているデータベース・エンジンです。 SQLite は、 すべての携帯電話とほとんどのコンピューターに組み込まれており、 人々が毎日使用する無数の他のアプリケーションにバンドルされています。 詳しくはコチラ

SQLite ファイル形式は安定しており、クロスプラットフォームで下位互換性があり、開発者は 2050 年までそれを維持することを約束しています。SQLite データベース ファイルは、 システム間でリッチ・コンテンツを転送するためのコンテナーとして([1] [2] [3])、 そして、 データの長期アーカイブ形式として([4])、 一般的に使用されます。 1兆 (1e12) を超える SQLite データベースがアクティブに使用されています([5])。

SQLite のソース・コードはパブリック・ドメインで、誰でも自由に目的を問わず使用できます。
